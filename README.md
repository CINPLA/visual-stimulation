# visual-stimulation
Analysis for experiments with visual stimulation e.g. drifting gratings.


### Dependencies

- `python >=3.5`
- `numpy`
- `scipy`
- `matplotlib`
- `seaborn`
- `elephant`
- `neo`
- `quantities`
